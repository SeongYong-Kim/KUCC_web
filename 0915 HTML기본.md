# HTML 기본



## 기본문법

```
<strong></strong>  : 글자 진하게

<u></u>  : 밑줄

<h1></h1> : 제일 큰 제목, 숫자 커질 수 록 크기가 작아짐

<br> : 줄 바꿈, 닫히는 테그없음

<p></p> : 단락 표시할 때 사용함, 정해진 여백만큼만 띄워줌

<img src = "이미지 주소">
<img width = "100%" src = ""> width, src 이런게 다 속성임

<li></li> 리스트로 만들어줌.
<ul></ul>가 부모가 됨, li들을 묶어줌, 얘는 순서가 없음
<ol></ol>가 부모가 됨, li들을 묶어줌, 얘는 순서가 있음

<a href=""></a> 링크테그
```



## 기본구조

```
<!doctype html> html로 작성되었다는 것을 의미
<head>
	<title></title> : 웹페이지의 제목을 나타냄
	<meta charset = "utf-8"> utf-8 방법으로 나타내어 줌
</head> 웹페이지의 정보

<body></body> 본문
```
